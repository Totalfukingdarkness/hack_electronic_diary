# Взлом школьного дневника

Скрипты предназначены для работы с учебной информационной системой школы. Используя их можно исправить плохие оценки, убрать замечания, а так же добавить похвалу.

## Зависимости

- Какую версию Python использовать, зависит от используемой вами версии [Djangо](https://docs.djangoproject.com/en/4.0/faq/install/#what-python-version-can-i-use-with-django).
- Скрипт разработан на версии Python 3.10, следовательно для использования он подойдет.

#  Скрипты и из запуск

Для запуск через shell, потребуется:
1. Запустить shell командой в терминале:
```
python manage.py shell
```
2. Вставить скопированный текст из файла `scripts.py`.
3. Далее вы можете использовать скрипты из файла


## fix_marks

Его работа заключается в том что бы исправить плохие оценки 2 и 3, на 5.

В качестве аргумента принимает ФИО ученика.

Для его работы требуется написать в `shell`:
```Python
fix_marks('Смирнов Алексей Дмитриевич')
```


## remove_chastisements

Удаляет все замечания из дненивника.

В качестве аргумента, принимат ФИО ученика.

Для его работы требуется написать в `shell`:
```Python
remove_chastisements('Смирнов Алексей Дмитриевич')
```

## create_commendation

Формирует случайную положительный похвалу преподавателя по указанному предмету.

В качестве аргумента, принимат ФИО ученика.

Для его работы требуется написать в `shell`:
```Python
create_commedation('Смирнов Алексей Дмитриевич', 'Физкультура')
```
